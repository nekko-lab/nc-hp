# Nekko Cloud 公式サイトリポジトリ

## 本サイトは[Astro](https://astro.build/)を使用しています

- Node.js をインストールしてください
- 動かすのに必要なライブラリは`npm install`でお願いします

## 情報の追加について

- サイト内で使用する画像は`src/public`内へ格納してください<br>どこからでも呼び出すことができます
- 基本的には component にのっとった形での情報追加をお願いします
- 新しく項目を作成する際にははなるべく使いまわしやすい形での追加をお願いします
- CSS に関しては[TailWind CSS](https://tailwindcss.com/)を使用しています

## 連絡について

- 何か連絡があればリポジトリの Issue へお願いします
